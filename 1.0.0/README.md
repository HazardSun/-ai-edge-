## 快速开始

### 安装插件

**Edge 浏览器：**

1. 打开 Edge，在地址栏输入 edge://extensions 并回车
2. 打开左下角「开发者模式」（开关勒处）
3. 点击「加载解压缩的扩展」
4. 选择本插件文件夹（相对于本文件所在的目录）
5. 安装成功后右上角会显示翻译图标

**Chrome 浏览器：**

1. 打开 Chrome，地址栏输入 chrome://extensions
2. 打开右上角「开发者模式」
3. 点击「加载已解压缩的扩展」
4. 选择插件目录

> **注意：** 如果你前一次已加载过插件，重新加载后请在 edge://extensions 页面点击插件卡片上的「刷新」按钮，使修改生效。

### 第一次使用

安装后首先配置一个可用的 API：
1. 点击右上角的翻译图标 → 点击右上角齿轮按钮
2. 或 右键图标 → 「扩展选项」
3. 选择一个提供商，填写必要的 API Key
4. 点击「测试连接」确认可用
5. 点击「保存配置」

## 配置 API

### 打开设置页

三种方式打开配置页：
- 右键浏览器右上角的插件图标 → 「扩展选项」
- 点击弹窗右上角的齿轮按钮 ⚙️
- 地址栏输入 edge://extensions → 点击插件卡片上的「详细信息」 → 「扩展选项」

### 连接设置

配置页包含 **3 张卡片**：

**① 接口地址 (API URL)**
- 网址格式：https://.../v1/chat/completions
- 可以点击下方的快速选择按钮一键填写

**② API 密钥 (API Key)**
- 从服务商网站获取
- 点击密码框右侧的眼睛图标可切换显隐
- 本地服务（Ollama / LM Studio）不需要 API Key，留空即可

**③ 模型名称 (Model ID)**
- 留空则使用服务端默认模型
- 下方的模型标签可点击快速填写（如 gpt-4o、deepseek-chat）

### 测试连接

配置完成后点击底部的「测试连接」按钮，成功后会显示响应时间和模型名称。然后再点击「保存配置」。

## 使用方法

### 方式一：选中文本翻译（推荐）

1. 在网页中用鼠标选取任意文本
2. 选中后会在文本旁边显示一个蓝色的翻译图标
3. 点击该图标，立即显示翻译结果浮窗

> 浮窗支持点击「复制」复制翻译结果，点击「重试」重新翻译，按 Esc 关闭浮窗。

### 方式二：弹窗输入翻译

1. 点击浏览器右上角的插件图标
2. 在弹出的窗口中输入要翻译的文本
3. 选择目标语言（默认为简体中文）
4. 点击「翻译」或按 Ctrl + Enter
5. 翻译结果显示后可点击复制按钮复制

> 点击弹窗右上角的齿轮 ⚙️ 可直接跳转到配置页。

### 方式三：右键菜单翻译

1. 在网页中选中文本
2. 右键鼠标
3. 点击「翻译选中文本」
4. 立即显示翻译结果浮窗

### 方式四：快捷键 (Alt+T)

1. 在网页中选中文本
2. 按 Alt + T
3. 立即显示翻译结果浮窗

## 支持的服务商

| 服务商 | API URL (预设) | 需要 API Key | 说明 |
|----------|-----------|-------------|------|
| **OpenCode** | https://opencode.ai/zen/v1/chat/completions | 是 | 默认服务商 |
| **DeepSeek** | https://api.deepseek.com/v1/chat/completions | 是 | 性价比高 |
| **OpenAI** | https://api.openai.com/v1/chat/completions | 是 | GPT-4o / GPT-4o-mini |
| **Moonshot** | https://api.moonshot.cn/v1/chat/completions | 是 | 月之暗面 |
| **Ollama (本地)** | http://localhost:11434/v1/chat/completions | 否 | 本地部署，免费 |
| **LM Studio (本地)** | http://localhost:1234/v1/chat/completions | 否 | 本地部署，免费 |

> 也支持其他任意兼容 OpenAI API 格式的服务商（如 Qwen、GLM、Azure OpenAI 等），只需在设置中手动输入对应的 API URL 即可。

## 外观设置

在配置页的「外观设置」选项卡中可以调整：

- **主题**：自动跟踪系统 / 浅色 / 深色
- **字体大小**：滑块调整翻译浮窗的字体（10~24px）

> 设置保存后即生效，无需重新加载网页。

## FAQ / 常见问题

### Q：点击翻译后显示"无法连接到 API 服务器"

**原因：** API URL 不正确或服务未启动。
**解决：**
1. 打开设置页，检查 API URL 是否正确
2. 如果是本地服务（Ollama / LM Studio），确认服务已启动
3. 点击「测试连接」确认能正常访问

### Q：显示"API 请求失败 (HTTP 401)"

**原因：** API Key 无效或已过期。
**解决：**
1. 到服务商网站重新生成一个 API Key
2. 在插件设置中更新并保存

### Q：显示"无法从 API 响应中解析翻译结果"

**原因：** API 返回的数据格式不符合预期。
**解决：**
1. 确认 API URL 是否正确
2. 查看服务商的文档，确认兼容 OpenAI 格式

### Q：浮动图标没有出现

**原因：** 可能是当前网页限制了 content script。
**解决：**
1. 尝试刷新页面
2. 仍无效则使用右键菜单或快捷键 Alt+T

### Q：快捷键 Alt+T 没反应

**原因：** 可能与其他扩展或系统快捷键冲突。
**解决：**
1. 打开 edge://extensions
2. 点击右上角「键盘快捷键」
3. 找到"翻译选中文本"修改为其他组合

### Q：支持 Chrome 吗？

支持。安装方法同上，地址栏输入 chrome://extensions 即可。
